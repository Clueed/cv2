# CV / Resume

A Typst resume with data imported from `data.typ`. Supplementary context for tailoring (job highlights, additional interests, skills, etc.) lives in the wiki — start at `../wiki/projects/cv2/cv2.md`.

## Build

```bash
make preview          # from resume/
# or: make preview    # from repo root — delegates here
```

**Produces:** `dist/main.pdf` and `dist/page-N.png` (300ppi) inside `resume/`.

**Review:** View the PNG with the Read tool (it can read images directly).

**Use only `make preview` for compiling.**
